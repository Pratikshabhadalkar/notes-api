# Notes API 🚀

A simple REST API built using Node.js and Express.

## Features
- Create Note (POST)
- Get All Notes (GET)
- Delete Note (DELETE)
- Status codes used (200, 201, 400)

## Tech Stack
- Node.js
- Express
- Postman (for testing)

## How to Run

1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Server runs on:
   http://localhost:3000

## API Endpoints

GET /notes  
POST /notes  
DELETE /notes/:id
